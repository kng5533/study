# 青色申告用の会計ソフト - 要件定義書

# 1 概要

- 青色申告用の会計ソフトのアプリケーション開発

# 2 目的
・毎年の確定申告の効率を図るため

# 3 想定ユーザーと役割

- 毎回終わるまでに、5時間かけて手入力している仕分けを一部自動化する。（目標1時間）
- PayPayは取引履歴からCSVがダウンロードできるから、取引日、取引内容（店名・摘要）、金額を取得
- ガソリン費の勘定科目を「旅費交通費」として、記帳
- 旅費交通費 〇〇〇 円 / 普通預金 〇〇〇円
- 仮払消費税 〇〇円   /

# 4 主な機能一覧

# 4-1 認証/アカウント管理
・ログイン（DBに直接ユーザー登録するから、登録画面は不要）
・ユーザーID、パスワードでログイン

# 4-2 データ登録
・CSVアップロード（PayPay取引履歴からCSV）
・手動入力で取引追加
・日付、摘要、借方、貸方、金額の登録
・仮払／仮受消費税の自動計算
・勘定科目

# 4-3 自動仕訳
・店名や摘要ベースで分類
・手動で修正したらルールに記憶させる

# 4-4 表示・集計

・取引一覧（借方・貸方・金額・税額・状態）
・未分類一覧
・月次合計や消費税の集計
・PDF出力（青色申告用）
